    PDF Extract for a sample PDF File

    It's a sample application for a coding competition.

    It's a PDF Extract software to extract four components from a sample pdf file as in the folder. It extracts four things from the PDF File :- A. Reg, No. B. Logo or Image Text. C. Company Name. D. Address.

    All the above four are populated into database and Reg, No. is marked as a primary key.
